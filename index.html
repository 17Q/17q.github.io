17Q Repository

Current Version 1.0.1

Instructions for adding the 17Q repository in Kodi:

  • In Kodi, click on the Settings Gear
  • Click on "File manager"
  • Click on "Add source"
  • In the top box, enter the path for the source as https://17q.github.io/
  • In the bottom box name the media source as .17Q Repository and click ok
  • Back to the main menu click "Add-ons"
  • Next click the "box" icon at the top to enter the Add-on browser
  • Click Install from zip file
  • Next click the source name ".17Q Repository" and then select repository.17q-1.0.1.zip
  • Select OK and in a few seconds you will get a confirmation the repo was installed
  • Next click Install from repository and select the 17Q Repository
  • Select the add-ons you wish to install
